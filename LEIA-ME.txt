📋 O QUE FOI CRIADO?
═══════════════════════════════════════════════════════════════

Um aplicativo completo para criar listas de compras compartilhadas
que sincroniza em tempo real entre todos os dispositivos da família!


✅ FUNCIONALIDADES:
───────────────────────────────────────────────────────────────

✓ Adicionar itens com quantidade
✓ Marcar itens como comprados
✓ Excluir itens da lista
✓ Limpar todos os itens comprados de uma vez
✓ Sincronização em tempo real entre dispositivos
✓ Funciona offline
✓ Pode ser instalado no celular como app
✓ Interface moderna e fácil de usar
✓ SEM necessidade de login/cadastro
✓ Totalmente GRATUITO

⚡ INÍCIO SUPER RÁPIDO (TL;DR)
═══════════════════════════════════════════════════════════════

1. Abra o terminal nesta pasta

2. Execute:
   npm install

3. Configure o Firebase:
   - Edite o arquivo: src/firebase.js
   - Siga as instruções no arquivo

4. Execute:
   npm run dev

5. Abra no navegador:
   http://localhost:5173/

PRONTO! ✅
